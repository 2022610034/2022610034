1. head: Tag ini menandai awal dari bagian kepala (head) dari halaman HTML. Bagian ini berisi informasi yang tidak ditampilkan langsung di halaman, seperti metadata dan pengaturan halaman

2. meta charset="UTF-8">: Metadata ini memberitahu browser bahwa karakter set yang digunakan adalah UTF-8, yang mendukung berbagai karakter, termasuk karakter internasional.

3. meta name="viewport" content="width=device-width, initial-scale=1.0">: Metadata ini digunakan untuk mengatur tampilan halaman pada perangkat berbasis web. Pengaturan ini memberitahu browser agar menyesuaikan lebar halaman dengan lebar perangkat (device-width) dan mengatur tingkat zoom awal (initial-scale) menjadi 1.0.
