Laporan

1. Pendahuluan

1.1 Latar Belakang
File sharing is the activity of sharing or providing data access to digital media on the internet to other people. The data shared can be very diverse, ranging from computer programs, documents, electronic books, or multimedia such as images, video, and audio. However, to be able to share with each other, the file provider must upload the file to the server first. Then other people can download it from that server.

Adel Ali Al-Zebari states that most of today's internet traffic is due to peer-to-peer file sharing systems. In the research he has done, he created an application to share files between mobile devices using J2ME and MIDP technology peer to peer on computers and mobile devices. In this research, problems were found in the form of the ability to download larger files, making this application very slow.

1.2 Rumusan Masalah

1.2.1 Tujuan Penelitian
Tujuan dari penelitian ini adalah untuk mengidentifikasi dan menganalisis masalah yang ada dalam sistem berbagi file peer-to-peer yang ada saat ini, serta untuk mengembangkan solusi yang lebih efisien dan cepat.

2 Landasan Teori

2.1 Studi Literatur

Berbagai studi literatur telah dilakukan dalam domain file sharing dan jaringan peer-to-peer. Penelitian sebelumnya telah menyoroti berbagai masalah yang muncul dalam aplikasi berbagi file saat ini, termasuk keterbatasan dalam ukuran file yang dapat didownload dan kecepatan transfer data yang lambat.

2.2 Teori Pendukung

Teori pendukung yang mendasari penelitian ini meliputi prinsip-prinsip jaringan peer-to-peer, algoritma transfer file, serta konsep keamanan informasi dalam berbagi file secara online.

3 Metodologi

Penelitian ini menggunakan metode penelitian kualitatif, di mana data dikumpulkan melalui studi literatur, observasi, dan wawancara dengan pengguna sistem berbagi file.
